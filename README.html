Guia da Professora — Publicando no GitHub Pages

  1. Crie um repositório, por exemplo: Desenvolvimento-mobile.
  2. Envie estes arquivos para a branch main.
  3. Em Settings → Pages, selecione Deploy from a branch e escolha main / root.
  4. Acesse o site publicado e compartilhe o link com os alunos.

Como replicar para outras UCs

Duplique o repositório e ajuste: título, plano de semanas e os arquivos em lessons e /assignments.

Alternativa ao Android Studio

  • PWA: este site já vem configurado com manifest e Service Worker para instalar como app.
  • Expo (Snack & Expo Go): prático para testar recursos nativos no celular sem instalar Android Studio. Os alunos instalam o app Expo Go no celular e acessam o projeto via QR Code ou link.
