01. Introdução a JavaScript para Mobile

Vamos alinhar a base de JavaScript pensando em apps móveis. Semelhanças para quem vem de Python:

  • Variáveis (let, const) e tipos dinâmicos
  • Funções (declaração e arrow)
  • Arrays, objetos e JSON
  • DOM, eventos e fetch para chamadas HTTP

Exemplo prático

// Somatório simples
function soma(a,b){ return a+b }
console.log(soma(2,3))
