04. Web APIs de Dispositivo

Use recursos do dispositivo via navegador (quando suportado):

  • Câmera: getUserMedia
  • Geolocalização: navigator.geolocation
  • Vibração: navigator.vibrate
  • Clipboard, Orientação, Battery, etc.
// Exemplo: geolocalização
navigator.geolocation.getCurrentPosition(pos => {
  console.log(pos.coords.latitude, pos.coords.longitude);
});
