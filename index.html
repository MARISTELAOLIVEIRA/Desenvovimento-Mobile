Mobile first com JavaScript: do Web App ao PWA e Expo

Nesta unidade, você vai construir apps móveis sem depender do Android Studio. Começamos com fundamentos de JavaScript, avançamos para apps web responsivos e transformamos em PWA. Depois, exploramos Expo (via Expo Go e Snack) para acessar recursos nativos quando necessário.

Começar agora Guia da Professora
Plano de 8 semanas (exemplo)
  1. JS Essentials para Mobile: Sintaxe, DOM, eventos, fetch
  2. UI Responsiva: Grid/Flex, componentes, acessibilidade
  3. Estado & Armazenamento: localStorage, padrões simples
  4. PWA: manifest, service worker, offline & install
  5. APIs de Dispositivo (Web): câmera, geolocalização, vibração
  6. Expo (Snack & Expo Go): sem Android Studio
  7. Integração com API: REST, autenticação básica
  8. Projeto Final: PWA ou App Expo
Trilhas & Materiais
01. Introdução a JavaScript para Mobile 02. UI Responsiva: Grid, Flex, componentes 03. PWA: manifest, service worker & offline 04. Web APIs de Dispositivo (Câmera, Geo, etc.) 05. Expo sem Android Studio (Snack & Expo Go) 06. Projeto Final: PWA ou Expo
Entregas (GitHub Classroom)
  • Checkpoint 1: JS Básico + DOM
